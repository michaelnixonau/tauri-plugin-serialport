## Default Permission

Allows using serialport functionality

- `allow-available-ports`
- `allow-cancel-read`
- `allow-close`
- `allow-close-all`
- `allow-force-close`
- `allow-open`
- `allow-read`
- `allow-write`
- `allow-write-binary`
